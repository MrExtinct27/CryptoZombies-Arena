   → Click "Connect Wallet"
   → Confirm in MetaMask

2. Create Zombie
   → Enter name
   → Confirm transaction
   → Show in "View Army"

   → Select 2 zombies
   → Name the offspring
   → Confirm transaction

   → Select your zombie
   → Choose opponent
   → Attack & see result

   → List zombie for sale
   → Browse market
   → Switch account & buy (optional)

   → View all players
   → Click on entry for details
   → Show "YOU" badge


🐛 TROUBLESHOOTING
──────────────────────────────────────────────────────────────

Port Already in Use:
  lsof -ti:8545 | xargs kill -9
  npm run dev

Clean Restart:
  killall node
  rm -rf build/
  npm run dev

MetaMask Reset:
  Settings → Advanced → Clear Activity Tab Data

Hard Refresh Browser:
  Mac: Cmd + Shift + R
  Windows: Ctrl + Shift + R
